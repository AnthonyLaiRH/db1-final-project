# CSI2132 - Project

# Setup
Add JDBC postgres jar file to repo folder

# To Run
In the main repo directory run the follow commands
### Windows
javac src\cli\SQL_CLI.java
java -cp postgresql-42.2.11.jar src\cli\SQL_CLI.java
### Bash
javac src/cli/SQL_CLI.java
java -cp postgresql-42.2.11.jar src/cli/SQL_CLI.java
